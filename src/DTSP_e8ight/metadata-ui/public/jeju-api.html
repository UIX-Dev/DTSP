AirQuality 공기질 데이터 API

▶
GET /api/airquality 전체 공기질 데이터 조회 ▶

Parameters

Name Type Required Description
page integer 페이지 번호 (기본값: 0)
size integer 페이지 크기 (기본값: 20)
stationId string 측정소 ID

Responses

200 성공

Parameters

Name Type Required Description

Name Type Required Description
